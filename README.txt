Preferred System Requirements:
RAM - 12/16 GB
Screen Resolution - 1920 * 1080
Screen Scaling - 100%


Installation Procedure:
1. Copy the "Pac-Man" folder to your PC. Suppose you selected "D:" drive.

2. Boot up the PacSetup file. Go to the "Select Installation Folder" page. Click on "Browse" and select the Pac-Man folder which you just copied to your pc. Example: "D:\Pac-Man\".

3. After installation is finished, you will see a "Pacman.exe" file. Open it and enjoy the game.
